# Virtual-XP-Desktop
Browser frontend with a fake retro Desktop Environment with several apps (Calculator, Media Player, Browser, Games, Bf Interpreter)


Features:
- Notepad
- Paint Image Editor
- Media Player (audio/video)
- CSV Spreadsheet Creator
- Basic Calculator
- Music Visualiser with Audio Mod Features (Load audio file and Modify Pitch/Bass/Speed/Reverb)
- Brainf*ck / Lambda Calculus IDE and Interpreter
- Internet Browser (Majority of websites will not load due to X-Frame-Options policy)
- Games:
  - Solitaire
  - Conway's Game of Life
  - Minesweeper
  - Tetris
  - Blackjack
  - Baccarat
  - 5 Card Poker
- Computer Generated 3D World Scenes (City, Spaceship, Ocean, Crystal Cavern)
